# Task 1 - Scan Local Network for Open Ports

## Objective
Discover open ports on devices in the local network.

## Tools Used
- Nmap
- Windows Command Prompt

## Steps Performed
1. Installed Nmap.
2. Found local IP using ipconfig.
3. Performed TCP SYN scan.
4. Identified open ports.
5. Analyzed possible security risks.

## Outcome
Learned basic network reconnaissance and open port identification.
